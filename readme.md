# Umbraco Practitioners front-end development

Front-end development work for the Umbraco Practitioners website, using Gulp, Nunjucks and Tailwind CSS.

| Package  | Version |
| :------- | :------ |
| node     | 15.5.1  |
| npm      | 7.3.0   |
| yarn     | 1.22.10 |
| gulp-cli | 2.3.0   |
| gulp     | 4.0.2   |

## Installation
1. Clone the repository
2. Run `yarn` to install packages
3. Run `gulp`
